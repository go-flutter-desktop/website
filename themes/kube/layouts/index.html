Hover

Hover brings Flutter apps to the desktop.

Getting started View on Github

Cross-platform

Hover works on all modern desktop operating systems: Windows, MacOS and Linux.

Expanding on Flutter

Hover uses Flutter tooling whenever possible, to keep things simple and recognizable.

Plugin support

Existing plugins can be used in combination with Hover.

Hot reload

Hover uses the existing flutter tooling to provide hot reload. Just hover run.

Open-Source

Hover is licensed under BSD 3-clause license.
Hover is free for personal and commercial use.

Golang

Hover uses go-flutter, a Go/GLFW embedder for Flutter, to bring your app to the desktop.
